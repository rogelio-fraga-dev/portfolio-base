Full Stack Application

Sistema de Gestão Clínica - Full Stack

Sistema completo de gestão clínica desenvolvido como aplicação full-stack, integrando frontend em Angular com backend em NestJS. Solução moderna e intuitiva para administração de clínicas médicas, com arquitetura bem estruturada que separa responsabilidades entre frontend e backend.

Ver Código no GitHub

Funcionalidades Principais

  • Gerenciamento de Pacientes: Cadastro completo com histórico médico, documentos e informações de contato, permitindo busca e filtros avançados.
  • Sistema de Agendamentos: Interface intuitiva para criação, edição e cancelamento de consultas, com validação de horários e disponibilidade.
  • Prontuário Eletrônico: Registro digital de consultas, diagnósticos, prescrições e evolução do paciente com histórico completo.
  • Dashboard Administrativo: Painel com métricas e estatísticas em tempo real para tomada de decisão.
  • Autenticação e Segurança: Sistema de login seguro com JWT e controle de acesso baseado em perfis (administrador, médico, recepcionista).
  • Interface Responsiva: Design adaptável que funciona perfeitamente em desktop, tablet e dispositivos móveis.

Arquitetura e Tecnologias

Frontend: Angular, TypeScript, RxJS, Angular Router, HTML5, CSS3
Backend: NestJS, TypeScript, JWT, class-validator, class-transformer
Ferramentas: Git, GitHub, Node.js, npm

Backend Architecture

Backend para Gestão Clínica - Microserviços

Plataforma de backend robusta para um sistema de gestão de clínicas, utilizando arquitetura de serviços distribuídos. Este projeto foi concebido com foco em práticas de nível de produção, abordando segurança, governança e observabilidade completa para garantir uma fundação sólida, manutenível e de alta performance.

Ver Código no GitHub

Características Arquiteturais

  • Arquitetura Orientada a Serviços: Sistema modularizado em serviços independentes por domínio de negócio (administrativo e atendimento), promovendo separação de responsabilidades e facilitando manutenção.
  • API Gateway & Service Discovery: Centraliza roteamento e segurança com Spring Cloud Gateway, enquanto Netflix Eureka permite registro e descoberta dinâmica dos serviços, garantindo resiliência.
  • Segurança com JWT: Sistema completo de autenticação e autorização com Spring Security. Após login com senhas criptografadas (BCrypt), um JWT é gerado para autorizar acesso seguro aos endpoints.
  • Observabilidade Completa: Saúde e performance de cada serviço monitoradas em tempo real com Prometheus e visualizadas em dashboards no Grafana. Rastreamento de requisições com OpenTelemetry e Jaeger permite depuração precisa de gargalos.
  • Containerização: Todos os serviços containerizados com Docker para facilitar deploy e escalabilidade.
  • Banco de Dados: Integração com MySQL para persistência de dados com configuração otimizada para alta performance.

Stack Tecnológico

Backend: Java 17, Spring Boot 3, Spring Cloud, Spring Security
Infraestrutura: Docker, MySQL, Netflix Eureka, Spring Cloud Gateway
Observabilidade: Prometheus, Grafana, Jaeger, OpenTelemetry
Ferramentas: Maven, Git, GitHub
